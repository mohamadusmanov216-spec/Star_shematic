# SBuild

> **Fabric-мод для Minecraft 1.21.8** для автоматизированного строительства по схематикам.

SBuild — это архитектурная база для будущего AI-бота строителя:
- загрузка `.litematic` схем,
- анализ материалов,
- работа с сундуками-хранилищами,
- стратегии установки сложных блоков,
- redstone-анализ и валидация цепей,
- bot-runtime для автоматизированных действий.

---

## ✨ Что уже есть

- Модульная архитектура (`schematic`, `materials`, `storage`, `block`, `redstone`, `bot` и др.)
- Базовый entrypoint Fabric и bootstrap инициализации модулей
- Командный root `/sbuild`
- Система placement-стратегий для сложных блоков
- Подсистема redstone-графа и проверки схем

---

## 🧩 Технологии

- **Minecraft**: `1.21.8`
- **Loader**: Fabric
- **Language**: Java 21
- **Build**: Gradle + Fabric Loom
